工程英语视听说使用说明

发表于 2020-03-26 更新于 2020-03-31 Valine：
本文字数： 622 阅读时长 ≈ 1 分钟

准备

  1. 使用火狐Firefox，请直接阅读安装步骤的火狐条目

  2. 使用Chrome内核浏览器(360安全浏览器、360极速浏览器均可，新版Edge（chrome内核）)【注意：由于Google Chrome安全策略升级，70以上版本无法离线安装crx扩展】​

  3. CRX扩展文件 点我下载 [若链接失效请通过反馈方式与我联系]适用于Chrome内核

安装

  • Chrome内核(以360为例)
    1. 打开360浏览器,将CRX文件拖入浏览器窗口，弹窗点击确定
    2. 打开网页工程英语视听说脚本
    3. 点击安装此脚本，等待几秒钟
    4. 点击安装即可
    5. 进入到视听说网站，到答题界面。可以看到右边的窗口即代表安装成功。
  • 火狐浏览器(FireFox)
    1. 安装插件 添加到FireFox,确认.
    2. 剩下步骤与Chrome内核相同 点击阅读

使用

  • 答题
    1. 脚本会自动匹配答案并选择或填充答案，无需手动选择。
    2. 点击自动提交，会在填写后自动提交答案，无需手动提交。只需点击页面上Next，秒秒钟完成一单元作业。
  • 刷时长
    1. 选择时长：有10、30、60分钟提供选择。
    2. 刷新频率：每一分钟刷新页面一次，防止掉线同时刷在线时长。
    3. 点击开始刷时长【注意：刷时长过程中无法答题，可以通过点击停止刷时长以继续答题】。
    4. 刷时长界面如下：
    5. 界面会有“刷时长中”提示，点击停止刷时长便可继续答题。下方记录已刷新次数和时间。

常见问题

  1. 360浏览器成功安装后，未出现助手界面；

    解决：将浏览器内核改为极速内核（即Chrome内核）
